# ECS171 Project: Income Prediction

##  Description
This project predicts annual household income based on six key social and demographic features using a machine learning model. 

## Features Used for Prediction
- Education Level  
- Occupation  
- Location  
- Work Experience  
- Employment Status  
- Gender
